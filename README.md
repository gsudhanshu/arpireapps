Prerequisites: php 7.4, composer 2

1. git clone https://github.com/gsudhanshu/arpireapps.git
2. composer install
3. php artisan passport:keys
4. php artisan passport:install
5. php artisan migrate
6. php artisan db:seed
7. php artisan serve
